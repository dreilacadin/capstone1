Synopsis: - Bootstrap Overview - Files Structure - Grid System - Components What is Twitter Bootstrap? - Boostrap is a powerful mobile first front-end framework for faster and easier web development. - It uses HTML, CSS and JavaScript. History - Boostrap was developed by Mark Otto and Jacob Thornton at Twitter. It was released as an open source product in August 2011 on GitHub. Why use Bootstrap? - Mobile first approach: -Since Bootstrap 3, the framework consists of Mobile first styles throughout the entire library instead of in separate files. - Browser Support: -It is supported by all popular browsers. Why use Bootstrap? - Easy to get started - Responsive design What Bootstrap Package Include? - Scaffolding: Basic structure with Grid System, link styles, background. - CSS: Global CSS settings, fundamental HTML elements styled and enhanced with extensible classes, and an advanced grid system. - Components: Over a dozen reusable components build to provide iconography, dropdowns, navigation, alerts, popovers, and much more. - Javascript Plugins: Over a dozen custom jQuery plugins. You can easily include them all, or one by one. - Customize: You can customize Bootstrap's components, LESS variables, and jQuery plugins to get your very own version. Bootstrap Versions: -Bootstrap 2 - release on January 31 2012 -Bootstrap 3 - August 19, 2013 -Bootstrap 4 - beta version - Aug 10, 2017 http://getbootstrap.com/
